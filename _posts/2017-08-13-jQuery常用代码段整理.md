#### 平滑滚动
```
$('a').on('click', function (e) {
  e.preventDefault();
  var hash = this.hash;
  var target = $(hash);
  if (target.length > 0) {
    $('html,body').animate({
      'scrollTop': target.offset().top
    }, 400, function () {
      window.location.hash = hash;
    });
   }
});
```
