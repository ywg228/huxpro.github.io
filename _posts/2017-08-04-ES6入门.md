## ES6开发环境搭建
使用Babel将ES6编译成ES5。
- npm全局安装Babel（可打开WebStorm左下方的Terminal命令行或cmd打开命令行工具切换到当前项目目录)
```
npm install -g babel-cli
```
- 初始化项目
```
npm init -y
```
-y代表全部默认，不用一次次按回车了，完成后可以看到目录下生成package.json文件

- 本地安装Babel
```
npm install --save-dev babel-cli babel-preset-es2015
```
安装完后，可以查看package.json中的devDependencies。
```
 "devDependencies": {
    "babel-cli": "^6.24.1",
    "babel-preset-es2015": "^6.24.1"
  }
```

- 创建.babelrc配置文件

在项目根目录下新建.babelrc文件，打开后输入如下代码：
```
{
  "presets": [
    "es2015"
  ],
  "plugins": []
}
```
- 使用-o转换命令
```
babel src/index.js -o dist/index.js
```
这样你就可以看到dist文件夹下输出index.js了

- 如果你不想每次都重复这段命令
在package.json中
```
 "scripts": {
    "build": "babel src/index.js -o dist/index.js"
  },
```
这样使用下面就行了：
```
npm run build
```

## let声明变量
用let声明的变量只在花括号包裹的块中有用，即作用域只在块中，外部访问不到
``` 
for (let i = 0; i < 10; i++) {
}
console.log(i); // ReferenceError: i is not defined
``` 

注意：设置循环变量的那部分是一个父作用域，而循环体内部是一个单独的子作用域。它们不在同一个作用域，有各自单独的作用域。
``` 
for (let i = 0; i < 10; i++) {
   var let i = 666;
   console.log(i); //输出10个666
} 
``` 

#### 不存在变量声明

let在声明前不能被使用，所以在使用前一定要先声明。
``` 
console.log(i); //ReferenceError
let i = 10;

console.log(j); //undefined
var j = 2;
``` 

#### 暂时死区
- ES6中会对let声明的变量形成一个块级作用域，只在花括号包裹的块中有效，外部不能访问。
- 使用let声明变量之前，该变量都是不可用的，这称为“暂时性死区”。（temporal dead zone，简称 TDZ）。
- 使用let声明变量时，只要变量在还没有声明完成前使用，就会报错。

``` 
if (true) {
  // TDZ开始
  i = 10; // ReferenceError
  console.log(i); // ReferenceError

  let i; // TDZ结束
  console.log(i); // undefined

  i = 20;
  console.log(i); // 20
}
``` 
- ES6 规定暂时性死区和let、const语句不存在变量提升，主要是为了减少运行时错误，防止在变量声明前就使用这个变量，从而导致意料之外的行为。这样的错误在 ES5 是很常见的，现在有了这种规定，避免此类错误就很容易了。

#### 不允许重复声明 
let不允许在同一作用域内，重复声明同一个变量。
``` 
let i = 10;

let i = 20; // 报错
``` 
不能在函数内部重新声明参数。它们都在同一作用域
``` 
function func(arg) {
  let arg; // 报错
}
``` 

## 块级作用域
- ES5 只有全局作用域和函数作用域，没有块级作用域。

#### 为什么需要块级作用域？ 
块级作用域解决了ES5中常见的两个问题。 <br>
第一种情况：内层变量可能会覆盖外层变量。

```
 var i = 10;
 function func() {
   console.log(i);
   if(false) {
      var i = 20;  
   }
 }
 func(); //undefined
```

第二种情况：用来计数的循环变量泄露为全局变量 
```
for(var i = 0; i < 10; i++) {
  
}
console.log(i); //10
```

#### let为 JavaScript 新增了块级作用域
```
let i = 10;
if (true) {
  let i = 20;
}
console.log(i); // 10
```

#### 立即执行函数没有必要了
```
// IIFE 写法
(function () {
  var i = 10;
}());

// 块级作用域写法
{
  var i = 10;
}
```

## const声明常量
- const声明一个只读的常量。一旦声明，常量的值就不能改变。
- const一旦声明变量，就得赋值。
- 作用域与let一样：只在声明所在的块级作用域内有效，不存在变量提升，同样存在暂时性死区，只能在声明的位置后面使用。
```
const PI = 3.1415;
PI = 3; // TypeError: Assignment to constant variable.
```

## 解构赋值
#### 数组解构赋值
```
let [a, b, ...rest] = [1, 2, 3, 4, 5];
console.log(a, b, rest); //1 2  Array [3,4,5]
```

#### 对象解构赋值
```
let {a, b} = {a: 1, b: 2};
console.log(a, b); //1 2
```
不存在则为undefined
```
let {a, b, c} = {a: 1, b: 2};
console.log(a, b, c); //1 2 undefined
```
默认值
```
let {a, b, c=3} = {a: 1, b: 2};
console.log(a, b, c); //1 2 3
```
#### 字符串解构赋值
```
let [a, b] = 'hello';
console.log(a, b); //"h" "e"
```
#### 函数参数解构赋值
```
function add([x, y]){
  return x + y;
}
console.log(add([1, 2])); // 3
```

#### 使用场景
1. 交换变量的值
```
let x = 1;
let y = 2;
[x, y] = [y, x];
console.log(x, y);
```
2. 函数返回多个值
```
//返回数组
function func() {
  return [1, 2, 3];
}
let [a, b, c] = func();
console.log(a, b, c); //1 2 3
//返回对象
function func2() {
  return {i:1, j:2};
}
let {i, j} = func2();
console.log(i, j); //1 2
```

3. 函数参数定义
```
//有序参数
function func([x, y]) {
 console.log(x, y); //1 2
}
func([1, 2]);
//无序参数
function func({x, y}) {
  console.log(x, y); //1 2
}
func({y:2, x:1});
```
4. 提取JSON数据
```
let json = {
  name: "ywg",
  age: 18,
  arr: [1, 2, 3]
}
let {name, age, arr} = json;
console.log(name, age, arr);
```

5. 设置函数参数默认值
```
 function func(value, isValue=true, callback = function(){}) {
    console.log(value, isValue);
 }
 func('hello');
```
避免了在函数体内部再写var foo = config.foo || 'default foo';这样的语句。

6. 遍历Map结构
```
let map = new Map();
map.set('name', 'ywg');
map.set('age', 18);
for(let [key, value] of map) {
  console.log(key + ' = ' +value); //"name ywg" "age 18"
}
```
只获取键值
```
for(let [key] of map) {
  console.log(key);
}
```
只获取键名
```
for(let [,value] of map) {
  console.log(value);
}
```

## 函数扩展
#### 参数默认值
1. 通常情况下，我们把默认参数放在尾部。
```
function add(x, y=20) {
  console.log(x+y);
}
add(10); //30
add(10, 30); //40
```
2. 与解构赋值结合使用
```
function add({x, y=20}) {
  console.log(x, y);
}
add({}); //undefined 20
add({x:10}); //10, 20
add({x:10, y:30}); //1m 30
```
3. 如果默认参数不在尾部，这种情况默认参数不能省略，可传入undefined来使用默认值
```
function add(x, y=10, z) {
  console.log(x, y, z);
}
add(); //undefined 10 undefined
add(10, , 20); //报错
add(10, undefined, 20); //10 10 20
add(10, 20, 20); //10 20 20
```
4. length属性：默认参数和rest参数不会计入length属性。
```
(function (a) {}).length // 1
(function (a = 5) {}).length // 0
(function (a, b, ...rest) {}).length // 2
```
5. 设置了参数默认值，函数进行声明初始化时，参数会形成一个单独的作用域。等到初始化结束，这个作用域就会消失。
```
var x = 1;
function func(x, y = x) {
  console.log(y);
}
func(2); // 2
```

#### rest参数
用于获取函数的剩余参数，这样就不需要使用arguments对象了。rest参数的变量是一个数组。
```
function func(x, y, ...reset) {
  let num = 0;
  for(let i of reset) {
    num+= i;
  }
  console.log(num);
}
func(1, 2, 3, 4, 5);
```
使用rest参数替代arguments参数的例子
```
//数组排序
function sortNum() {
  return Array.prototype.slice.call(arguments).sort();
}
const sortNum = (...args) => args.sort();
console.log(sortNum(12,13,1,234,345));
```

**注意**：reset参数只能放在最后一位，后面不能跟有其它参数。否则会报错

#### 严格模式
只要函数参数使用了默认值、解构赋值、或者扩展运算符，那么函数内部就不能显式设定为严格模式，否则会报错。
```
//报错
function func(x, y=10) {
  'use strict';
}
```
可以设置全局严格模式，这样就不会报错了。
```
'use strict'; //放在顶部
....
function func(x, y=10) {
  
}
```

#### name属性
返回函数的函数名。
```
function func(){};
console.log(func.name);
```
如果将一个匿名函数赋值给一个变量，ES5 的name属性，会返回空字符串。
const func = function (){};
//ES5
console.log(func.name); //""
//ES6
console.log(func.name); //"func"

#### 箭头函数 重点
使用箭头 => 定义函数
1. 如果代码块只有一条语句，则可省略return和花括号
```
//ES5
var add = function(x, y) {
  return x + y;
}
//ES6
const add = (x, y) => x + y;
```
2. 如果没有参数或者多个参数，就使用圆括号代表参数部分,单个参数，可省略圆括号
```
const f = () => 5;
const f2 = n => 10;
```
3. 如果代码块多于一条语句，就必须要使用花括号括起来，并且使用return返回。
```
const add = (x, y) => {
  .....
  return x+y;
};
```
4. 与解构赋值结合
```
const func = ({x, y}) => x + ' ' +  y;
```
5. 使得代码更加简洁, 简化回调函数
```
const isEven = n => n%2==0;
```
```
// 正常函数写法
var result = values.sort(function (a, b) {
  return a - b;
});
// 箭头函数写法
var result = values.sort((a, b) => a - b);
```
6. 注意点
函数体内的this对象，就是定义时所在的对象，而不是使用时所在的对象。<br>
不可以当作构造函数，也就是说，不可以使用new命令，否则会抛出一个错误。<br>
不可以使用arguments对象。用 rest 参数代替。<br>
