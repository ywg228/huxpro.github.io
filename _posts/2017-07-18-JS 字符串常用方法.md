## 字符串反转
#### 方法1：最简单的
```
funtion reserveStr(str) {
  return str.spilt('').reverse().join('');
}
```
#### 方法2：栈
```
funtion reserveStr(str) {
  var arr = str.split('');
  var rs= []
  while(arr.length) {
    rs.push(arr.pop());
  }
  return rs.join('');
}
```
#### 方法3：for循环
```
funtion reserveStr(str) {
  var newStr = '';
  for(var i = str.length-1; i > 0; i--) {
    newStr += str.charAt(i); 
  }
  return newStr;
}
```
